Welcome to the home of Galaxy, the Meteor IRC bot!

Check out the source!

Commands

  • !mrt / !meteorite: Basic information about Meteorite
  • !atmo / !atmosphere: Basic information about Atmosphere
  • !pkg / !package <package>: Lookup an Atmosphere package
  • !commands / !list: List of Galaxy commands
  • !command / !help <command>: Lookup a Galaxy command, and provide information about it
  • !devs / !people: Basic information about the Meteor team
  • !np: Provides a "no problem" message when a user thanks you for your help
  • !dataja: "Don't ask to ask, Just ask!"
  • !ugt: Basic information about Universal Greeting Time
